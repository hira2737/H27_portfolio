## About

This is a **professional portfolio website** built with cutting-edge web technologies. It's designed to showcase my skills, projects, and expertise as a Computer Science student and aspiring web developer. The portfolio features smooth animations, modern design patterns, and an excellent user experience across all devices.

---

## Key Features

- **Responsive Design** вЂ“ Perfectly optimized for mobile, tablet, and desktop
- **Smooth Animations** вЂ“ Framer Motion-powered interactions for engaging UX
- **Dark/Light Mode** вЂ“ Toggle between themes with persistent preferences
- **Mobile-First Approach** вЂ“ Built with mobile users in mind
- **Accessible** вЂ“ WCAG compliant components and semantic HTML
- **Performance Optimized** вЂ“ Fast load times with Vite bundling
- **Component-Based** вЂ“ Modular architecture using shadcn/ui
- **Interactive Sections** вЂ“ Hero, About, Skills, Projects, Education, Certifications, Achievements, and Contact
- **Floating Animations** вЂ“ Custom animated tech icons in the hero section
- **Resume Download** вЂ“ One-click resume download functionality

## Installation & Setup

### Prerequisites
- **Node.js** 16+ 
- **npm** or **bun** package manager
- **Git**

### Clone Repository
```bash
git clone https://github.com/hira2737/H27_portfolio.git
cd H27_portfolio
```

### Install Dependencies
```bash
npm install
# or
bun install
```

### Development Server
```bash
npm run dev
# or
bun dev
```
The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
# or
bun run build
```

### Preview Production Build
```bash
npm run preview
# or
bun run preview
```

### Deploy to GitHub Pages
```bash
npm run deploy
# or
bun run deploy
```

## Key Features Explained

### Responsive Design
- Mobile-first approach using Tailwind CSS
- Breakpoints optimized for all screen sizes
- Fluid typography and spacing

### Smooth Animations
- Framer Motion for React components
- Page transition animations
- Scroll-triggered animations
- Hover effects and micro-interactions

### Dark/Light Mode
- System preference detection
- Manual theme toggle in navbar
- Persistent theme selection

### Accessibility
- Semantic HTML structure
- ARIA labels for interactive elements
- Keyboard navigation support
- Color contrast compliance

## Configuration Files

### `vite.config.ts`
