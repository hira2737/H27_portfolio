## About

This is a **professional portfolio website** built with cutting-edge web technologies. It's designed to showcase my skills, projects, and expertise as a Computer Science student and aspiring web developer. The portfolio features smooth animations, modern design patterns, and an excellent user experience across all devices.

---

## Key Features

- **Responsive Design** – Perfectly optimized for mobile, tablet, and desktop
- **Smooth Animations** – Framer Motion-powered interactions for engaging UX
- **Dark/Light Mode** – Toggle between themes with persistent preferences
- **Mobile-First Approach** – Built with mobile users in mind
- **Accessible** – WCAG compliant components and semantic HTML
- **Performance Optimized** – Fast load times with Vite bundling
- **Component-Based** – Modular architecture using shadcn/ui
- **Interactive Sections** – Hero, About, Skills, Projects, Education, Certifications, Achievements, and Contact
- **Floating Animations** – Custom animated tech icons in the hero section
- **Resume Download** – One-click resume download functionality

---

## Tech Stack

### Frontend Framework
- **React 18** – Modern UI library with hooks
- **Vite 5** – Lightning-fast build tool and dev server
- **TypeScript** – Type-safe JavaScript

### Styling & UI
- **Tailwind CSS 3** – Utility-first CSS framework
- **shadcn/ui** – High-quality, unstyled components
- **CSS Modules** – Scoped component styles
- **PostCSS** – CSS transformation tool

### Animations & Interactivity
- **Framer Motion** – React animation library
- **React Icons** – SVG icon library
- **Sonner** – Toast notifications

### Build & Deployment
- **Vite** – Fast build tool
- **GitHub Pages** – Static site hosting

### Development Tools
- **TypeScript** – Static type checking
- **ESLint** – Code quality
- **Vitest** – Unit testing
- **Playwright** – E2E testing

## Installation & Setup

### Prerequisites
- **Node.js** 16+ 
- **npm** or **bun** package manager
- **Git**

### Clone Repository
```bash
git clone https://github.com/hira2737/H27_portfolio.git
cd H27_portfolio
```

### Install Dependencies
```bash
npm install
# or
bun install
```

### Development Server
```bash
npm run dev
# or
bun dev
```
The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
# or
bun run build
```

### Preview Production Build
```bash
npm run preview
# or
bun run preview
```

### Deploy to GitHub Pages
```bash
npm run deploy
# or
bun run deploy
```

## Deployment

This portfolio is deployed on **GitHub Pages** at:
### [https://hira2737.github.io/H27_portfolio/](https://hira2737.github.io/H27_portfolio/)

### Deployment Steps
1. Build the project: `npm run build`
2. The build output goes to the `dist/` folder
3. Deploy command: `npm run deploy`
4. GitHub Pages automatically deploys from the `gh-pages` branch

## Key Features Explained

### Responsive Design
- Mobile-first approach using Tailwind CSS
- Breakpoints optimized for all screen sizes
- Fluid typography and spacing

### Smooth Animations
- Framer Motion for React components
- Page transition animations
- Scroll-triggered animations
- Hover effects and micro-interactions

### Dark/Light Mode
- System preference detection
- Manual theme toggle in navbar
- Persistent theme selection

### Accessibility
- Semantic HTML structure
- ARIA labels for interactive elements
- Keyboard navigation support
- Color contrast compliance

---

## Configuration Files

### `vite.config.ts`
Configures Vite with React and path aliases.

### `tailwind.config.ts`
Tailwind CSS customization for colors, fonts, and animations.

### `tsconfig.json`
TypeScript compiler options and path aliases.

### `vitest.config.ts`
Unit testing configuration.

### `playwright.config.ts`
End-to-end testing configuration.
